# ForestPulse Environmental Intelligence Dashboard

## Overview

ForestPulse is an interactive Power BI dashboard developed to monitor deforestation and land-use change across Malaysia.

The dashboard integrates multiple environmental datasets into a centralized platform to support decision-making through interactive visualizations and analytics.

---

## Features

- KPI Dashboard
- Interactive GIS Map
- Time-Series Analysis
- Fire Hotspot Monitoring
- CO₂ Emission Analysis
- Land Use Change Visualization
- Predictive Analytics
- What-If Simulation

---

## Technologies Used

- Power BI
- DAX
- Power Query
- Python
- Excel
- GIS

---

## Datasets

- Global Forest Watch
- MODIS Fire Alerts
- Global Carbon Project
- Land Use Change Dataset
